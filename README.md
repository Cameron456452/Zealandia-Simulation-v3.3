# Zealandia-Simulation-v3.3
Economic and Political Simulation for the fictional country of Zealandia. 

Spreadsheet URL:
https://docs.google.com/spreadsheets/d/1wa3ObdY3DZZ7mz6047Y4FY_YBduvY8CIIzxpmuE3M4Q/edit?usp=sharing

Tracks various economic statistics for the fictional nation of Zealandia, including economic and demographic data

Simulates elections for Zealandia based on demographic approval of each political party, can simulate any number of parties. The spreadsheet can currently handle up to over a dozen parties currently.
